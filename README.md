# Raspberry Book Reader

![raspberry book reader](./rpibookreader.jpg)

This is my second DIY project powered by Raspberry Pi and Arduino. It is a audiobook reader adapted to be used by the elderly.

## Features
1. Automatic switch on and off controlled by ATtiny85
2. Stopping and resuming playback
3. Rewinding or fast forwarding of the current playback
4. Switching track or chapter of a book
5. Switching book or album
6. Adjustable volume level
7. Optional audio messages of executed actions
8. Simple synthesizer of chapter and book numbers
9. Hot-spot network for remote device diagnostics
10. Reading content from a USB stick
